# Integration Examples of Xion network

This repository contains integration examples of Xion network.

## List of Examples

Here is the list of examples:

- [Todo App Web Demo](projects/todo-app-web-demo): A demo TODO WebApp that shows how to integrate with Xion network utilizing the `@burnt-labs/abstraxion` package.
- [Todo App Mobile Demo](projects/todo-app-mobile-demo): A demo TODO MobileApp that shows how to integrate with Xion network utilizing the `@burnt-labs/abstraxion-react-native` package.
- [Backend Session Demo](projects/backend-session-demo): A demo application that shows how to utilize backend session to integrate with Xion network in an existing Web2 application.
