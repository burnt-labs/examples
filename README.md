# Integration Examples of Xion network

This repository contains integration examples of Xion network.

## Libraries

- Xion.js: <https://github.com/burnt-labs/xion.js> - the library for interacting with the Xion network in TypeScript.
- Xion Types: <https://github.com/burnt-labs/xion-types> - the types definitions of all Xion Modules & Messages in different languages(Typescript, Python, Rust, etc.).

## List of Examples

Here is the list of examples:

- [Simple Web Demo](https://github.com/burnt-labs/xion.js/tree/main/apps/demo-app): A simple web demo that shows how to integrate with the Xion network using Xion.js.
- [Todo App Web Demo](https://github.com/burnt-labs/todo-app-frontend): A demo TODO WebApp that shows how to integrate with Xion network utilizing the `@burnt-labs/abstraxion` package.
- [Todo App Mobile Demo](https://github.com/burnt-labs/todo-app-expo-demo): A demo TODO MobileApp that shows how to integrate with Xion network utilizing the `@burnt-labs/abstraxion-react-native` package.
- [Backend Session Demo](https://github.com/burnt-labs/xion-backend-session-demo): A demo application that shows how to utilize backend session to integrate with Xion network in an existing Web2 application.
